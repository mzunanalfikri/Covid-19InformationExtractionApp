## Cara running

- Masuk ke dalam direktori src dan jalankan virtual environment dengan cara :
```
cd virtual/Scripts && activate && cd ../..
```
- Jalankan flask dengan cara :
```
flask run
```
- Buka alamat yang muncul di terminal.
